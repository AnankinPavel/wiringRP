# wiringRP
wiringPi for Repka Pi
## How to download wiringRP

```
# apt-get update
# apt-get install -y git
# git clone https://github.com/AnankinPavel/wiringRP.git
```

## How to build wiringOP

```
# cd wiringRP
# ./build clean
# ./build 
```


## How to uninstall wiringOP

```
# ./build uninstall
```
